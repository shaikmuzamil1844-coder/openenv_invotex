The Problem

AI Agents Break in Production

Today's AI agents are trained on static API schemas. But the real world is different — APIs change without warning.

1

API Schema Changes

A new required field appears silently in the endpoint.

2

Agent Gets 403 Error

The AI blindly retries the old format and fails repeatedly.

3

Business Impact

Refunds fail, customers are angry, revenue is lost.

Our Solution

Train AI to Adapt, Not Fail

🌍

Multi-Domain Environment

3 realistic enterprise domains — Email Triage, Traffic Control, Customer Support — with 9 tasks at 3 difficulty levels each.

💥

Schema Drift Mechanic

Dynamic 403 errors inject real API schema changes mid-episode. Standard models fail. Trained models adapt.

🧠

Unsloth Fine-Tuning

Llama-3 8B fine-tuned using SFT on successful environment trajectories. Training loss: 3.8 → 0.14 in 60 steps.

🎮

Interactive Playground

Live HuggingFace Space lets anyone test agents in real-time. Judges can interact directly with the environment.

📊

Automated Graders

Deterministic graders score every episode on accuracy and efficiency. Zero LLM-as-judge bias.

🐳

Production Ready

Fully containerized with Docker. Deployed on HuggingFace Spaces. Single environment variable to switch domains.

Technical Architecture

How It All Works

🔌

FastAPI Server

WebSocket-based step/reset endpoints. Pydantic validation on every tool call.

🗄️

SQLite + SQLAlchemy

Per-episode savepoints. Full rollback isolation. Swap to PostgreSQL with one env variable.

🧩

Domain Registry

Plugin system — each domain is an independent module implementing BaseDomain interface.

📏

Grader Pipeline

Post-episode evaluation by two graders: Task Completion + Efficiency Score.

⭐ Key Innovation

The Schema Drift Demo

Step 1: Agent tries refund normally

Tool: process_refund
Args: {"ticket_id": "TKT-9002", "amount": 390.0}

❌ API ERROR: 403 Forbidden — Schema
Validation Failed. The /v2/refunds endpoint
now requires an 'authorization_code'.

Step 2: Smart agent pivots and recovers

Tool: verify_identity → Gets AUTH-77X-BETA

Tool: process_refund + authorization_code

✅ Refund of $390.00 processed!
Reward: +0.15 | Done: progressing...

Why This Matters

❌

Standard Model (Qwen-72B)

Retries same call 3x. Gets stuck. Final score: 0.00

✅

Our Trained Llama-3

Detects error, pivots to verify_identity, retries with code. Final score: 0.65+

This is World Modeling — the agent detects that its mental model of the API is wrong, updates it in real-time, and recovers.
